# gw-proto

Проект `gw-proto` предназначен для хранения protobuf-файлов, которые используются для взаимодействия между двумя микросервисами: `gw-currency-wallet` и `gw-exchanger`. Эти файлы описывают структуру сообщений и сервисов, обеспечивая единый контракт для обмена данными между сервисами.

## Назначение

- Определение схем данных и сервисов с помощью Protocol Buffers (.proto файлов).
- Обеспечение совместимости и стандартизации взаимодействия между микросервисами.
- Упрощение генерации gRPC-клиентов и серверов на различных языках программирования.

## Использование

1. Разрабатывайте и обновляйте `.proto` файлы в данном репозитории.
2. Для генерации gRPC-клиентов и серверов используйте соответствующие инструменты protobuf и gRPC для вашего языка программирования. Например:

```bash
protoc --proto_path=./ --go_out=plugins=grpc:./generated ./path/to/your.proto
```

3. Интегрируйте сгенерированные файлы в ваши микросервисы `gw-currency-wallet` и `gw-exchanger` для реализации взаимодействия.

## Сборка

- Убедитесь, что у вас установлен `protoc` (Protocol Buffers Compiler).
- Установите необходимые плагины для генерации кода под ваш язык.
- Запустите команды генерации кода из `.proto` файлов.
- Интегрируйте сгенерированные файлы в ваши проекты.

Таким образом, `gw-proto` служит центральным хранилищем контрактов для взаимодействия между микросервисами, обеспечивая согласованность и удобство разработки.
